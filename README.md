# Nichinoken Test Analyzer

日能研のテスト結果画面からデータを一括抽出・パッケージングするためのChrome拡張機能です。

## 主な機能

| 機能 | 説明 |
|---|---|
| **📦 パッケージ一括取得** | テストの振り返りトップページからテスト一覧を自動検出し、選択したテストの正誤一覧+成績集計をZIPでダウンロード |
| **📊 正誤一覧抽出** | 問題別の正誤・正答率・単元情報をJSON構造化データとして取得 |
| **📈 成績集計抽出** | 得点・偏差値・順位の集計データを取得 |
| **🔍 分野別正答率** | 分野別正答率を全組み合わせで一括取得 |
| **🖼 画像ダウンロード** | Bookendビューアの問題用紙/解答用紙を画像でダウンロード |

## セットアップ方法

1. Chromeブラウザで `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトのディレクトリを選択

> **ローカルのMHTMLで試す場合**: 拡張機能の「詳細」→「ファイルの URL へのアクセスを許可する」をオンにしてください。

## 使い方

### パッケージ一括取得（推奨）

1. [MY NICHINOKEN](https://mynichinoken.jp) にログイン
2. 「テストのふり返り」ページを開く
3. 拡張機能のアイコンをクリック
4. 取得したいテストにチェックを入れる
5. 取得データ（正誤一覧 / 成績集計）を選択
6. 「📥 パッケージをダウンロード」をクリック

### 単体データ抽出

1. テスト詳細ページ（正誤一覧 or 成績）を開く
2. 拡張機能のアイコンをクリック
3. 自動的にデータを抽出
4. 「📥 JSONをダウンロード」をクリック

