# Nichinoken Test Analyzer

日能研のテスト結果画面から苦手な単元を抽出・分析するためのChrome拡張機能です。

## 主な機能

| 機能 | 説明 |
|---|---|
| **データ取り込み** | 日能研のテスト結果ページから自動抽出＆ローカル保存 |
| **JSON構造化** | 実施日（学年/年/月/日）を構造化データとして保存 |
| **苦手ランキング** | 間違えた単元を全教科/教科別でランキング表示 |
| **要注意問題** | 全体正答率60%以上なのに×の問題を警告 |
| **苦手克服トラッカー** | 過去×→最新○の単元を自動検出・祝福 |
| **期間フィルタ** | 日付範囲を指定して表示・ダウンロード |
| **古い削除** | 指定日以前のデータを一括削除 |
| **JSON出力/復元** | ダウンロード＆インポートでOS移行にも対応 |

## セットアップ方法
1. Chromeブラウザで `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトのディレクトリを選択

> **ローカルのMHTMLで試す場合**: 拡張機能の「詳細」→「ファイルの URL へのアクセスを許可する」をオンにしてください。

## ファイル構成
- `manifest.json` — 拡張機能の設定（Manifest V3）
- `content.js` — ページからデータを抽出するスクリプト
- `popup.html / popup.js / popup.css` — ポップアップUI
- `README.md` — この説明書
